# Contributing to Bagisto Iyzico

Thank you for considering contributing to Bagisto Iyzico!

## How to Contribute

### Reporting Bugs

- Search existing issues first
- Open a new issue with PHP version, Bagisto version, and package version
- Provide minimal reproduction steps

### Submitting Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Test with a working Bagisto installation
5. Submit a pull request

## Development Setup

```bash
git clone https://github.com/yellow-three/bagisto-iyzico.git
cd bagisto-iyzico
```

Place the package in your Bagisto `packages/Webkul/Iyzico/` directory and register the service provider.

## Code Standards

- Follow PSR-12 coding style
- Use strict types: `declare(strict_types=1)`
- Keep methods focused and short
